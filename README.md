# Motywator-bot

To prosty bot motywacyjny napisany w Pythonie przez **bbonbeq**.

## Jak działa?

Co 10 sekund wyświetla losową wiadomość motywacyjną w terminalu.

## Jak uruchomić?

1. Zainstaluj Pythona (jeśli jeszcze nie masz).
2. Skopiuj plik `bot.py`.
3. W terminalu wpisz:

```bash
python bot.py
